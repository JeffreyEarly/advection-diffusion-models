#  ymin

Lower box bounds `ymin`.


---

## Declaration
```matlab
 self.ymin
```
## Returns
+ `ymin`  row vector of lower coordinate bounds

## Discussion

  `ymin` stores the lower coordinate bound for each state
  dimension. Use `-Inf` for an unbounded lower side.
