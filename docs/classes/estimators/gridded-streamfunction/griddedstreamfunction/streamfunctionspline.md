#  streamfunctionSpline

Fitted COM-frame mesoscale streamfunction spline.


---

## Discussion

  This spline evaluates $$\psi(\tilde{x},\tilde{y},t)$$ in the
  centered coordinates
  $$\tilde{x} = x - m_x(t)$$ and $$\tilde{y} = y - m_y(t).$$
