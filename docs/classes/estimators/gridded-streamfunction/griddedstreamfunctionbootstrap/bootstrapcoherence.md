#  bootstrapCoherence

Lazy scalar coherence diagnostics for each bootstrap replicate.


---

## Discussion

  `bootstrapCoherence` is a row vector of length `nBootstraps`.
  When coherence is unavailable, the vector is filled with `NaN`.
