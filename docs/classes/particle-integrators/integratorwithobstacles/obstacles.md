#  obstacles

Polygonal obstacles used for reflecting boundaries.


---

## Type
+ Class: `polyshape`

## Declaration
```matlab
 self.obstacles
```
## Returns
+ `obstacles`  array of reflecting `polyshape` obstacles

## Discussion

  `obstacles` stores the original non-overlapping `polyshape`
  objects supplied to the constructor.
