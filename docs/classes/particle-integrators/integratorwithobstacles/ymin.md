#  ymin

Lower coordinate bounds `ymin`.


---

## Type
+ Class: `double`

## Declaration
```matlab
 self.ymin
```
## Returns
+ `ymin`  `1 x 2` row vector of lower coordinate bounds

## Discussion

  `ymin` stores the lower bound for each coordinate. Use `-Inf` to
  indicate an unbounded lower side.
