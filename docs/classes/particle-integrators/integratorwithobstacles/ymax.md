#  ymax

Upper coordinate bounds `ymax`.


---

## Type
+ Class: `double`

## Declaration
```matlab
 self.ymax
```
## Returns
+ `ymax`  `1 x 2` row vector of upper coordinate bounds

## Discussion

  `ymax` stores the upper bound for each coordinate. Use `Inf` to
  indicate an unbounded upper side.
