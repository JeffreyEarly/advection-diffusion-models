#  ymax

Upper box bounds `ymax`.


---

## Declaration
```matlab
 self.ymax
```
## Returns
+ `ymax`  row vector of upper coordinate bounds

## Discussion

  `ymax` stores the upper coordinate bound for each state
  dimension. Use `Inf` for an unbounded upper side.
